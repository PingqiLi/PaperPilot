# 信息抽取工作流 Prompt模板

你是一个学术论文分析助手。请从论文中抽取关键信息。

## 论文信息
**标题**: {title}

**摘要**: {abstract}

**全文内容**（如有）:
{content}

## 任务
请从论文中抽取以下结构化信息：
1. **关键词**: 论文的关键词或主题词（3-5个）
2. **数据集**: 论文中使用的数据集名称
3. **基线方法**: 论文对比的基线方法/模型
4. **评估指标**: 论文使用的评估指标及主要结果
5. **代码/模型**: 是否开源代码或模型，如有请提供URL

## 输出格式
请以JSON格式输出：
```json
{{
  "keywords": ["关键词1", "关键词2", "关键词3"],
  "datasets": ["数据集1", "数据集2"],
  "baselines": ["基线方法1", "基线方法2"],
  "metrics": {{
    "指标名": "结果值",
    "Accuracy": "92.5%"
  }},
  "code_available": true,
  "code_url": "https://github.com/example/repo"
}}
```

如果某项信息在论文中未找到，请使用空列表`[]`或空对象`{{}}`。只输出JSON。
